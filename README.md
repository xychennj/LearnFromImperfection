# Learning from imperfection: GA‑KF‑CEEMDAN‑BiLSTM‑TPA
This repository provides the implementation code for the landslide displacement prediction framework presented in the accompanying paper. The hybrid model integrates genetic algorithm (GA)‑optimized Kalman filtering, CEEMDAN‑based modal decomposition, and a BiLSTM‑TPA predictor. The workflow is organized as follows:
Hyperparameter optimization: A genetic algorithm is employed to determine the two key noise covariance matrices of the Kalman filter, namely the measurement noise covariance and the process noise covariance.
Adaptive denoising: The raw displacement time series are filtered using the Kalman filter with the optimized hyperparameters to suppress high‑frequency environmental noise.
Modal decomposition: The denoised signal is decomposed by CEEMDAN into a set of intrinsic mode functions (IMFs), which are subsequently grouped by frequency bands into three physically interpretable components: a random component (high frequency), a periodic component (medium frequency), and a trend component (low frequency).
Component‑wise prediction: A bidirectional long short‑term memory network augmented with a temporal pattern attention mechanism (BiLSTM‑TPA) is applied to predict each component separately. The final displacement forecast is obtained by summing the predictions of the three components.
All code is written in Python and has been tested on the two case study landslides (Zhakoushi and Huoshitan). Users are encouraged to adapt the framework to similar resource‑constrained monitoring scenarios.
