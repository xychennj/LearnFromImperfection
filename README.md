# Learning from imperfection: GA‑KF‑CEEMDAN‑BiLSTM‑TPA
这个项目的目的是提供我一篇文章的模型代码，项目内容基于人工智能算法融合预测滑坡位移。具体实现逻辑如下：1、利用遗传算法优化了卡尔曼滤波算法中的观测噪声协方差和预测噪声协方差两组超参数。2、将原始位移监测数据输入获得最优超参数的卡尔曼滤波算法中进行去噪处理。3、利用CEEMDAN算法将去噪后的数据进行模态分解。4、将分解出来的多个IMFs根据频率段进行融合，具体融合为（频率由高到低）随机项、周期项和趋势项。5、利用BiLSTM-TPA模型分别预测随机项、周期项和趋势项位移值并相加，得到最终位移预测值。/The goal of this project is to provide the model code for a paper focused on predicting landslide displacement using hybrid AI algorithms. The specific implementation logic is as follows: First, the Genetic Algorithm is utilized to optimize two sets of hyperparameters within the Kalman Filter algorithm, namely the measurement noise covariance and the process noise covariance. Second, the raw displacement monitoring data is input into the Kalman Filter algorithm that has obtained the optimal hyperparameters to undergo denoising processing. Third, the CEEMDAN algorithm is employed to perform modal decomposition on the denoised data. Fourth, the multiple decomposed IMFs are integrated according to frequency bands, specifically being fused into a random component, a periodic component, and a trend component (ordered from high frequency to low frequency). Finally, the BiLSTM-TPA model is utilized to separately predict the displacement values of the random, periodic, and trend components, which are then summed to obtain the final predicted displacement value.
