# Learning from imperfection: GA‑KF‑CEEMDAN‑BiLSTM‑TPA
The goal of this project is to provide the model code for a paper focused on predicting landslide displacement using hybrid AI algorithms. The specific implementation logic is as follows: First, the Genetic Algorithm is utilized to optimize two sets of hyperparameters within the Kalman Filter algorithm, namely the measurement noise covariance and the process noise covariance. Second, the raw displacement monitoring data is input into the Kalman Filter algorithm that has obtained the optimal hyperparameters to undergo denoising processing. Third, the CEEMDAN algorithm is employed to perform modal decomposition on the denoised data. Fourth, the multiple decomposed IMFs are integrated according to frequency bands, specifically being fused into a random component, a periodic component, and a trend component (ordered from high frequency to low frequency). Finally, the BiLSTM-TPA model is utilized to separately predict the displacement values of the random, periodic, and trend components, which are then summed to obtain the final predicted displacement value.
